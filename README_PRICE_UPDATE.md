# Обновление цен без доступа к Google Drive

## Что изменено
- Интерфейс сохранён: `bon_prices_local.html` — обновлённая версия вашего `bon.html`.
- Основная база услуг остаётся в `db.js`.
- Цены можно обновлять отдельно через `prices.js`.
- Пользователям не нужен доступ к вашему Google Drive.

## Как использовать
В одной папке должны быть:
```
bon_prices_local.html
db.js
prices.js
```
Открываете `bon_prices_local.html`. Если `prices.js` есть, цены берутся из него. Если его нет — приложение использует старые цены из `db.js`.

## Как менять цены из Google Sheets
1. В Google Sheets оставьте 2 колонки: `code` и `price`.
2. Скачайте таблицу как CSV.
3. Преобразуйте CSV в `prices.js` в формате:
```js
const PRICES = {
  "18": 905,
  "19": 1073
};
```
4. Замените старый `prices.js` новым.

## Важно
В `prices.js` можно хранить не все услуги, а только изменённые цены. Например:
```js
const PRICES = {
  "18": 950,
  "2214": 120
};
```
Остальные цены останутся из `db.js`.
