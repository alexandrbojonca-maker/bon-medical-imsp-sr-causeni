AUTO-SYNC GOOGLE SHEETS

Structura tabelului:
code | name | unit | price | quantity

Pași:
1. Deschideți tabelul în Google Sheets.
2. File → Share → Publish to web.
3. Alegeți foaia necesară și format CSV.
4. Copiați linkul.
5. În program, lipiți linkul în câmpul Google Sheets CSV link.
6. Apăsați Conectează.

După aceea programul actualizează catalogul la deschidere.
