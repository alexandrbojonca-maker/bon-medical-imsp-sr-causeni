CATALOG MANUAL PENTRU MEDICAMENTE/MATERIALE

Fișier de lucru: catalog_manual_template.csv

Coloane obligatorii:
- name / denumire
- price / pret

Coloane recomandate:
- code = cod intern, ex. EXT-0001
- unit = unitate, ex. cutie, fiolă, buc
- quantity = cantitate/stoc

Cum se folosește:
1. Deschide catalog_manual_template.csv în Excel.
2. Adaugă sau modifică medicamente/materiale.
3. Salvează ca CSV.
4. În aplicație apasă la „Catalog manual CSV” și încarcă fișierul.
5. Pozițiile apar automat în căutare și în tabul Farmacia.

Datele încărcate se salvează în browser (localStorage).
